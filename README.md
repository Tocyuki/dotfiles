# dotfiles

My dotfiles

## Feature

- some development tools
- vim
- tmux
- fish

## Supported OS

- Mac
- Ubuntu

## Run

### Installation

```bash
git clone https://github.com/Tocyuki/dotfiles.git
cd dotfiles
make init-{mac,ubuntu}
make deploy
```

### Usage

```bash
make help
```
